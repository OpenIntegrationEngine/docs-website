# Documentation site

## Prerequisites

1. [Node](https://nodejs.org/en/download) of some sort
   - _I'm using v22, but I'm sure v24 works too_
2. [Bun](https://bun.sh/)

## Usage

Run the development script for hot reloads
```sh
bun run docs:dev
```
RUn the build script fo static ocmpilation
```sh
bun run docs:build
```
